# A1 - Project and Package Definition

## Project Identity
- Title: Physics-Guided Operational Stiffness Proxies for Reduced-Order Screening of Harmonic Resonance Risk in 33/11 kV Distribution Substations
- Package name: `resonance-risk-screening`
- Import name: `resonance_risk_screening`
- Python: `>=3.10,<3.13`
- Build backend: `hatchling`
- Package manager: `uv`

## Scope
- In scope:
  - Reduced-order screening of resonance susceptibility from routine operational substation data.
  - Physics-guided proxy computation, operating-state reconstruction, and ordinal risk modeling.
  - Manuscript artifact generation (7 tables, 7 figures).
  - Interactive pipeline entrypoint for non-expert workflows.
- Out of scope:
  - EMT-grade harmonic simulation and waveform-level harmonic measurement campaigns.
  - Closed-loop real-time utility control deployment.

## Objectives and Acceptance IDs
- ACC-OBJ-01: Compute physically interpretable proxy indicators from routine measurements.
- ACC-OBJ-02: Reconstruct representative operating regimes using PAM medoid clustering.
- ACC-OBJ-03: Implement reduced-order susceptibility formulation grounded in resonance physics.
- ACC-OBJ-04: Build interpretable ordinal probabilistic risk model (low/moderate/high).
- ACC-OBJ-05: Demonstrate robustness and practical utility via temporal validation and benchmarks.

## Data and Governance Constraints
- Data class: operational grid data, `PII = NONE`.
- Data governance: utility-internal data path plus open benchmark for reproducibility fallback.
- Reproducibility: deterministic pipeline configuration and run-level logging.

## User Profile and Accessibility
- Target user: beginner-coder and analyst users in utility/research settings.
- Mandatory non-coder access: guided CLI pipeline.
- Error style: user-friendly messages with corrective hints.

## Observability Requirements
- Structured logging required: `true`.
- Log formats: `jsonl` + human-readable text.
- Default log directory: `logs/`.
- Required run/statistics schema fields are defined in proposal and enforced by pipeline logger.
