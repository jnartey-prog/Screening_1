# A4 - Data Handling and Test Plan

## Data Handling Rules
- Enforce explicit schema for required fields (timestamp, busbar voltage, incomer current, feeder currents/power).
- Timestamp harmonization required before any feature computation.
- Missing data policy: configured interpolation method with audit log of imputed fractions.
- Outlier policy: MAD filter with thresholds recorded in run logs.
- No PII accepted; reject unexpected identifier fields.
